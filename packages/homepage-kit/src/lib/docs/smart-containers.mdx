# Smart Containers

Smart Containers are AdorableCSS's revolutionary approach to creating perfect container elements with minimal code. The `64x64` syntax isn't just width and height - it's a complete layout solution.

## What Makes Them "Smart"?

Traditional CSS requires multiple properties to create a well-behaved container:

```css
.container {
  width: 64px;
  height: 64px;
  display: flex;
  align-items: center;
  justify-content: center;
  overflow: hidden;
  position: relative;
}

.container img {
  width: 100%;
  height: 100%;
  object-fit: cover;
  object-position: center;
}
```

With Smart Containers, this becomes:

```html
<div class="64x64">
  <img src="avatar.jpg" />
</div>
```

## Syntax Variations

Smart Containers support any valid CSS unit:

```html
<!-- Pixels (default) -->
<div class="64x64">...</div>

<!-- Percentages -->
<div class="100%x50%">...</div>

<!-- Mixed units -->
<div class="200pxx4rem">...</div>

<!-- Viewport units -->
<div class="50vwx100vh">...</div>
```

## vs. Basic Dimensions

Understanding when to use Smart Containers vs. basic dimensions:

### Use Smart Containers (`64x64`) when:
- Creating image containers
- Building icon buttons
- Making profile pictures
- Designing thumbnails
- Any container that needs centering

### Use Basic Dimensions (`w(64) h(64)`) when:
- Building form inputs
- Creating pure layout containers
- You need specific positioning behavior
- Working with non-centered content

## Performance Benefits

Smart Containers provide:

- **Reduced CSS bundle size**: One utility instead of 5-7 classes
- **Better caching**: Consistent patterns across your app
- **Fewer reflows**: Optimized for modern browser rendering
- **Predictable behavior**: Same centering logic everywhere

## Migration Guide

Converting existing containers to Smart Containers:

```html
<!-- Before -->
<div class="w(64) h(64) display(flex) align-items(center) justify-content(center) overflow(hidden)">
  <img src="avatar.jpg" style="object-fit: cover;" />
</div>

<!-- After -->
<div class="64x64">
  <img src="avatar.jpg" />
</div>
```

The Smart Container approach saves you **75% of the CSS classes** while providing **more functionality** out of the box.
