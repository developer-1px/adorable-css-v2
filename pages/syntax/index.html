Parser Rules

Separators

  • / - Multiple values
  • + - Combined values
  • : - Key-value pairs
  • .. - Ranges/gradients
  • . - Opacity notation

Prefixes

  • hover: - Hover state
  • md: - Responsive
  • ! - Important
  • group- - Group state
  • .. - Range prefix

Special Values

  • auto - Auto keyword
  • none - None keyword
  • full - 100% token
  • 16:9 - Aspect ratio
  • 100%-20 - Calc
