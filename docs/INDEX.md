## 🔄 PARA 관리 프로세스

### 정기 리뷰 일정
- **Projects**: 매주 금요일
- **Areas**: 매월 첫째 주
- **Resources**: 분기별
- **Archive**: 반기별 (6월, 12월)

### 문서 생애주기
```
Ideas → Projects → Areas → Resources → Archive
```

### 분류 기준
- **"이것이 특정 결과물로 이어지는가?"** → Projects
- **"이것을 지속적으로 관리해야 하는가?"** → Areas  
- **"나중에 참고할 가치가 있는가?"** → Resources
- **"더 이상 활성화된 관심사가 아닌가?"** → Archive
