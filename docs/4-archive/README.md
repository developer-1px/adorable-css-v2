## 📁 구조

### ✅ completed-projects/
**목적**: 완료된 프로젝트들의 최종 결과물

- 프로젝트 완료 보고서
- 결과물 링크 및 문서
- 학습된 교훈 (Lessons Learned)

### 🗂 deprecated/
**목적**: 더 이상 사용하지 않는 문서들

- 구버전 가이드
- 변경된 프로세스의 이전 버전
- 레거시 시스템 문서
