# Quick Start Guide

Get up and running with AdorableCSS in just a few minutes.

## Installation

Install AdorableCSS via npm:

```bash
npm install adorable-css
```

Or use it directly from CDN:

```html
<script src="https://unpkg.com/adorable-css/dist/adorable.min.js"></script>
```

<h2 id="first-component" class="heading(h2) mt(12) mb(6)">Your First Component</h2>

Let's create a simple card component using AdorableCSS:

```html
<div class="card p(6) bg(white) shadow(md) r(xl)">
  <div class="hbox gap(4) items(start)">
    <div class="size(64) r(full) bg(purple-500)"></div>
    <div class="vbox gap(2) flex(1)">
      <h3 class="font(lg) bold c(gray-900)">Hello AdorableCSS!</h3>
      <p class="text(sm) c(gray-600)">
        This is your first component using Figma-inspired syntax.
      </p>
      <button class="button(primary) mt(2)">Learn More</button>
    </div>
  </div>
</div>
```

<h2 id="core-concepts" class="heading(h2) mt(12) mb(6)">Understanding the Syntax</h2>

AdorableCSS uses a function-based syntax that maps to CSS properties:

<div class="grid(2) gap(4) my(6)">
  <div class="card p(4)">
    <h4 class="font(medium) mb(2)">Layout</h4>
    <ul class="vbox gap(1) text(sm)">
      <li><code class="docs-code">hbox()</code> - Horizontal flexbox</li>
      <li><code class="docs-code">vbox()</code> - Vertical flexbox</li>
      <li><code class="docs-code">gap(4)</code> - Spacing between items</li>
    </ul>
  </div>
  
  <div class="card p(4)">
    <h4 class="font(medium) mb(2)">Styling</h4>
    <ul class="vbox gap(1) text(sm)">
      <li><code class="docs-code">c(gray-900)</code> - Text color</li>
      <li><code class="docs-code">bg(white)</code> - Background</li>
      <li><code class="docs-code">p(6)</code> - Padding</li>
    </ul>
  </div>
</div>
